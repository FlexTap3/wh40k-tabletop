# Meta Practice Pack — real GT-winning lists from 11th edition's opening weeks

Five import-ready army lists for `wh40k-tabletop.html`. File 00 is **your** list (the Iron Hands GT winner archetype from the Army Guide); 01–04 are the top meta lists to spar against — all real lists that won (or defined the archetype at) the first 11th-edition GTs, June 20–27 2026.

## How to import
1. Open the app → Army tab → **Import List**.
2. Paste the entire contents of one of the `.txt` files (faction is auto-detected from the header lines).
3. Tick "deploy" if you want the tokens placed immediately.
For a solo practice game: host in one browser window, join from a second, and import your list in one and a meta list in the other.

## Notes
- Wargear detail lines are omitted — the app pulls profiles from its own DB; unit names and model counts are what the importer matches (verified by simulating the importer against the app DB: every unit matches, with one exception below).
- **List 04:** the Navigator (75pts) is an Imperial Agents *allied* unit, so the importer (which searches only the detected faction) will report it "not found" — add it manually from the Army tab under Imperial Agents after importing.
- Enhancements/detachment rules are name-only in the app by design; the source lists' enhancements are noted in `~/WH40k/Notes/11th Edition Tournament Meta - Living Notes.md` and the Army Guide.
- Source articles: Goonhammer Competitive Innovations in 11th — "New Era-dication" pt.1–2 and "Eleven Times Eleven" pt.1–3.
