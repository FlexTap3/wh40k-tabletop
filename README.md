# WH40k Tabletop

A fan-made, self-contained virtual tabletop for Warhammer 40,000 (11th edition):
peer-to-peer online play (PeerJS), a solo mode against a tactical AI, army builder,
dice, terrain layouts and rules aids — all in one HTML file.

**Play it: https://flextap3.github.io/wh40k-tabletop/**

Contains game-functional data only (names, stats, points). No Games Workshop
rules text, lore or card prose. Not affiliated with Games Workshop.
